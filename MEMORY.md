## Data And Lexicons

- N5 and N4 lexicons are loaded from `data/jlpt-n5-words.js` and `data/jlpt-n4-words.js`.
- N4 chapters are generated by kana order with a maximum of 40 words per chapter.
- Tatoeba text examples may be used with attribution in `docs/data-sources.md`.
- First-version reading uses system TTS, not Tatoeba audio.

## Validation Notes

- Reliable checks in this repo:
  - `node --check learner.js`
  - `node --check shared.js`
  - `npm run validate:lexicons`

- Browser automation may fail in the Codex sandbox. If so, rely on parse checks plus local HTTP checks and be conservative about visual claims.
- If the page becomes blank after edits, suspect JavaScript syntax or encoding damage first.
